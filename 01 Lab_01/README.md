# Lab 01 — C++ Basics

Small exercises completed as part of coursework, covering core C++ language features.

| File                  | Topic                                                                               |
|   ---                 |    ---                                                                              |
| `01_input_output.cpp` | Reading input and displaying output using `cin`/`cout`                              |
| `02_macro.cpp`        | Macro expansion — implementing a `SQUARE` macro without built-in math functions     |
| `03_namespace.cpp`    | Namespaces — defining multiple functions with the same name in different namespaces |
| `04_implement.cpp`    | Output formatting using `endl`, `setw`, `setprecision`, `fixed`, and `scientific`   |                        |                                                                                     |

## How to Compile and Run

```bash
g++ 01_input_output.cpp -o 01_input_output
./01_input_output
```

(Replace the filename for each exercise; on Windows use `.exe` instead of running with `./`)
