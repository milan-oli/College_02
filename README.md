# College_02: Object Oriented Programming (C++) Lab Practices

Welcome to the **College_02** repository. This repository contains the complete set of laboratory exercises and programming assignments for the **Object Oriented Programming (ENCT 151)** course, structured in accordance with the official Institute of Engineering (IOE) Bachelor's Degree syllabus.

The primary objective of these practices is to build a robust foundation in C++ and master the core pillars of Object-Oriented Programming (OOP).

---

## 📚 Course Overview

* **Course Title:** Object Oriented Programming
* **Course Code:** ENCT 151
* **Year/Part:** I / II
* **Language Used:** C++

---

## 🛠️ Repository Structure & Lab Syllabus

This repository is organized into **9 Laboratory Exercises** encompassing the entire course content, culminating in a final programming project.

### 📁 [Lab 01] Basics of C++ & Input/Output Handling
This directory contains solutions for the introductory assignments focusing on fundamental C++ syntax, basic I/O streams, preprocessor directives, custom namespaces, and stream manipulators.

#### 📝 Problem Statements:
1. [cite_start]**Basic Stream I/O:** Write a program to read inputs from a keyboard and display the output on the monitor using standard Input/Output streams (`cin` and `cout`). [cite: 181]
2. [cite_start]**Macro Expansion:** Implement a preprocessor macro in C++ called `SQUARE` that takes an argument and returns the square of that number using macro expansion techniques, without relying on any built-in mathematical functions. [cite: 182, 183]
3. [cite_start]**Namespaces Implementation:** Create a program demonstrating the utility and scope of namespaces. [cite: 184]
   * [cite_start]Define three distinct namespaces: `English`, `Nepali`, and `Newa`. [cite: 184]
   * [cite_start]Implement a `greet()` function inside each namespace that returns a greeting message in its respective language. [cite: 185]
   * [cite_start]Invoke and display the output of each `greet()` function from the `main()` function. [cite: 186]
4. [cite_start]**Stream Manipulators:** Write a program to demonstrate the precise layout and formatting control of output data using standard stream manipulators: `endl`, `setw`, `setprecision`, `fixed`, and `scientific`. [cite: 187]
