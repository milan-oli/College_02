# Lab 03 — Pointers, References, and Intro to Classes

Small exercises completed as part of coursework, covering return-by-reference, pointers with arrays, structures, and an introduction to classes including constructors.

| File                      || Topic                    |
|---                        ||---                       |

| `01_return_by_reference.cpp` || Returning a reference from a function — modifying the original variable through the function's return value |

| `02_pointer_with_array.cpp` || Using a pointer to traverse and access elements of an array |

| `03_structure.cpp` || `BankAccount` structure with deposit and withdraw methods, including insufficient balance handling |

| `04_class_teacher.cpp` || `Teacher` class with a member function defined outside the class using the scope resolution operator (`::`) |

| `05_constructor.cpp` || `Person` class demonstrating a parameterized constructor and a user-defined copy constructor |

## How to Compile and Run

```bash
g++ 01_return_by_reference.cpp -o 01_return_by_reference
./01_return_by_reference
```

(Replace the filename for each exercise; on Windows, run the generated `.exe` instead)
