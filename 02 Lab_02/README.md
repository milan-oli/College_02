# Lab 02 — Functions, Overloading, and Memory Management

Small exercises completed as part of coursework, covering function overloading, inline functions, default parameters, pass-by-reference, and dynamic memory allocation.

| File || Topic |

|---   ||---    |

| `01_volm_using_overloading.cpp` || Function overloading — calculating volume of a cube, cuboid, and cylinder |

| `02_swap_values.cpp` || Function overloading — swapping two values for int, char, float, and string using pass-by-reference |

| `03_largest_of_two_using_inline_fun.cpp` || Inline function to find the largest of two numbers |

| `04_calculate_discounted_price.cpp` || Function with a default parameter value (discount percentage) |

| `05_update_salary.cpp` || Pass-by-reference — updating a value inside a function so changes persist in the caller |

| `06_new_delete_operetor.cpp` || Dynamic memory allocation using `new`/`delete` for both single variables and arrays |

## How to Compile and Run

```bash
g++ 01_volm_using_overloading.cpp -o 01_volm_using_overloading
./01_volm_using_overloading
```

(Replace the filename for each exercise; on Windows, run the generated `.exe` instead)
